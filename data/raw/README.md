# Raw Data Staging

Place unmodified public datasets here for local experiments. Do not commit raw
traces or derived dataset dumps.

Expected RanSAP layout for `analysis/ae_public_eval/config.json`:

```text
data/raw/RanSAP/
  dataset/
    original/
    extra/
```

Use the source repository or Kaggle archive referenced in
`docs/evidence/ransap_public_data_note.md`, then keep the files local.
