# Draft Paper

This directory contains a provisional XeLaTeX manuscript for the
storage-embedded ransomware anomaly-detection research.

Build from the repository root:

```sh
make
```

or from this directory:

```sh
make
```

The generated PDF is written to `paper/build/main.pdf`.

The manuscript is intentionally framed as a hypothesis and evaluation-plan
paper. It must not be read as an empirical-result paper until reproducible
experiments and MNN memory/parity measurements are added.
